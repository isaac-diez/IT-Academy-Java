- #### English
Given the file.txt containing countries and capitals. The program has to read the file and store the data in a HashMap. The program asks for the user's name, and then has to display a random country, stored in the HashMap. The user must type the name of the capital of the country in question. If the user guessed right, one point is awarded. This action is repeated 10 times. Once you have randomly requested the capital of 10 countries, it must be stored in a file called stats.txt with the user's name and the score.
